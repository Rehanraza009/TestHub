# 🚀 TestHub

**TestHub** – A Modern Online Examination System built with HTML, CSS, JavaScript & Firebase

---

## 📌 About Project

TestHub is a smart and responsive Online Examination Platform designed for:

* Schools
* Colleges
* Coaching Centers
* Placement Tests

It provides separate dashboards for Admin and Students with secure login, exam creation, and result management.

---

## ✨ Features

### 👨‍💼 Admin Panel

* Add / Manage Exams
* Add Questions
* Manage Students
* View Results
* Dashboard Analytics

### 👨‍🎓 Student Panel

* Secure Login / Register
* Attend Exams
* View Results
* Edit Profile
* Mobile Responsive Dashboard

### 🔥 Extra Features

* Firebase Authentication
* Realtime Database
* Beautiful UI Design
* Fast Performance
* Responsive (Mobile + Desktop)
